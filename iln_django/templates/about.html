Digitization Details

The images were captured using the library's Phase One PowerPhase FX camera. The images are stored as 300 dpi tiffs, and displayed as jpeg images and gif thumbnails.

The text is stored in an eXist database, and retrieved using queries managed by PHP, XSLT and CSS. The site is harvested by an OAI harvester to produce Dublin Core records for each article.
